#***So sánh loading với SSR và CSR
Dev k thể cố làm cho CSR như SSR được vì browser nó làm tự động ở bước này:
CSR: browser gửi request lấy html -> dữ liệu vô nghĩa k render luôn -> download JS -> render tạm loading -> fetch API -> render với data đủ
SSR: browser gửi request lấy html -> page hiện khoảng trắng -> dữ liệu được server fetch API xong r nên có nghĩa dù k thực hiện được chức năng nhưng vẫn render luôn -> download JS event file -> thực hiện được actions
Còn thực tế về mặt loading cũng kp là React hơn hẳn, thực tế nếu có nhiều data và trang web load lâu với lazy loading thì việc query tiếp từ server cũng mang lại tốc độ khá tốt dù khoảng trắng dài. Nhược điểm k thể tránh khỏi của SSR là server phải xử lý nặng hơn.
